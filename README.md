一个基于Vue 3 + TypeScript + Element Plus的事件倒计时应用。

## 功能特性

- 🕐 实时倒计时显示
- ➕ 添加新事件
- ✏️ 编辑现有事件
- 🗑️ 删除事件
- 💾 数据本地存储
- 📱 响应式设计
- 🎨 美观的用户界面

## 技术栈

- Vue 3
- TypeScript
- Element Plus
- Vite
- dayjs

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 部署到 GitHub Pages

### 手动部署

```bash
# 一键构建并部署到 GitHub Pages
npm run deploy
```
